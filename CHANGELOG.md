# Smart Crowd Management System

## Version 1.0

### Major Features

- Real-time camera monitoring
- YOLOv8 person detection
- ByteTrack multi-object tracking
- Real-time people counting
- Occupancy percentage calculation
- Capacity monitoring
- Risk assessment engine
- Audible alarm for critical occupancy
- Automatic evidence screenshot capture
- Dashboard alert visualization
- Event logging
- Camera health monitoring
- Optimized real-time dashboard
- Modular software architecture

---

## Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8
- ByteTrack
- Tkinter
- NumPy
- Pillow
- Pygame

---

## Current Status

The Smart Crowd Management System Version 1.0 provides real-time crowd monitoring, occupancy analysis, risk assessment, and automatic alert generation with evidence capture through a modular and scalable architecture.
